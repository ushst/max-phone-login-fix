# Политика приватности

Расширение «Вход по номеру для MAX (неофициально)» не собирает, не хранит
и не передаёт никакие данные пользователя, никуда за пределы страницы
`https://web.max.ru` не отправляет запросов и не использует аналитику,
трекеры или удалённый код.

## Что делает расширение

Content-script выполняется только на `https://web.max.ru/*` и:

- ищет на странице ссылку «Sign in with phone number»;
- снимает с неё CSS-ограничения (z-index/видимость), из-за которых её может
  перекрывать блок с QR-кодом;
- добавляет на страницу собственную кнопку-дублёр, клик по которой
  программно нажимает эту ссылку.

Никакого доступа к содержимому вводимых пользователем данных (номер
телефона, код из SMS и т.п.) расширение не получает и не читает — оно
только управляет видимостью и кликабельностью уже существующего элемента
интерфейса сайта.

## Разрешения

Расширение не запрашивает ни одного `permission` из manifest.json — только
`content_scripts` с `matches` на один конкретный домен. Оно не может читать
куки, localStorage других сайтов, историю, вкладки и т.д.

## Связь с MAX

Расширение не является официальным продуктом мессенджера MAX и никак не
аффилировано с его разработчиками. Это независимый инструмент для
удобства пользователей интерфейса.
