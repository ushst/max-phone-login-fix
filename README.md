# Вход по номеру для MAX (неофициально)

Браузерное расширение для `web.max.ru`. На странице входа по умолчанию
виден только QR-код, а ссылка «Войти по номеру телефона» — мелкая
подпись под ним, которую легко не заметить или которую перекрывает блок
с QR-кодом. Расширение делает эту ссылку гарантированно видимой и
кликабельной, дублируя её постоянной кнопкой внизу экрана.

Не связано с разработчиками MAX, не собирает и не передаёт никакие
данные. Подробности — в [PRIVACY.md](PRIVACY.md).

## Установка вручную (пока не опубликовано в Chrome Web Store)

1. Скачайте/склонируйте этот репозиторий.
2. Откройте `chrome://extensions`.
3. Включите «Режим разработчика».
4. «Загрузить распакованное расширение» → выберите папку репозитория.

Либо возьмите готовый `max-phone-login-fix.zip` из
[Releases](https://github.com/ushst/max-phone-login-fix/releases) и
распакуйте перед загрузкой (Chrome не принимает zip напрямую как
unpacked-расширение).
